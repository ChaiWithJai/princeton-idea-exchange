# Princeton Idea Exchange - Design System

## Color Palette

### Primary Colors
- **Background (cream)**: `#f8f6f3`
- **Primary Orange**: `#ff5900`
- **Primary Navy/Dark Blue**: `#001666`

### Text Colors
- **Dark Text**: `#2a3132`
- **Medium Gray**: `#5f6566`
- **Light Gray**: `#8a8f90`

### Accent Colors
- **Orange (CTA buttons)**: `#ff5900`
- **Navy Blue (headings)**: `#001666`

## Typography

### Font Families
1. **Libre Baskerville** (serif) - Used for headings and hero text
   - Italic style for main headlines
   - Weights: 400, 700

2. **DM Sans** (sans-serif) - Used for body text and navigation
   - Weights: 400, 500, 600, 700

3. **Fragment Mono** (monospace) - Used for code/technical text
   - Weight: 400

4. **Inter** (sans-serif) - Alternative body font
   - Weights: 400, 500, 600

### Font Sizes
- **Hero H1**: 48-80px (Libre Baskerville Italic)
- **Section H2**: 36-48px (Libre Baskerville Italic)
- **Card H5**: 24-28px (DM Sans Bold)
- **Body**: 16-18px (DM Sans Regular)
- **Small/Tags**: 12-14px (DM Sans Medium)

## Spacing System
- **XS**: 8px
- **SM**: 16px
- **MD**: 24px
- **LG**: 40px
- **XL**: 80px
- **2XL**: 160px

## Components

### Navigation
- Fixed header with transparent background
- Logo in center (text-based: "Princeton Idea Exchange")
- Works link on left
- Playground link on right
- Font: DM Sans, 16px

### Buttons
- **Primary CTA**: Orange (#ff5900) background, white text, rounded corners
- **Secondary**: Outlined with navy border
- Padding: 12px 24px
- Border-radius: 8px

### Cards (Case Study)
- Light cream background
- Rounded corners (16px)
- Shadow on hover
- Image at top
- Tags as pills with dark background

### Tags/Pills
- Background: Dark (#2a3132)
- Text: White
- Border-radius: 20px
- Padding: 6px 16px
- Font: DM Sans, 14px

### Section Layout
- Max-width: 1200px
- Centered content
- Large vertical gaps (80-160px between sections)

## Animation & Interactions
- Smooth scroll behavior
- Hover effects on links (color transition)
- Cards have subtle scale/shadow on hover
- Text animation on hero (letter-by-letter reveal)

## Page Structure

### Homepage Sections
1. **Navigation** - Fixed top
2. **Hero** - "AI Strategy Consultants" with animated subtext
3. **Featured Works Carousel** - Horizontal scroll with image cards
4. **Action, not strategy** - Two case study cards
5. **What we bring to the table** - Scrolling service tags + text
6. **Behind the curtain** - Dark section with team photos
7. **Testimonials** - Client quotes with photos
8. **Brand logos** - Scrolling marquee
9. **CTA Section** - Contact links
10. **Footer** - Logo + copyright

### Works Page
- Grid of case study cards (2 columns)
- Same CTA section as homepage

### Case Study Pages
- Hero with title and tags
- Featured image
- Article content (Challenge, Solution, Results)
- Blockquote for testimonial
- Navigation to other works

### Playground Page
- Grid of experimental project cards
- Minimal layout
